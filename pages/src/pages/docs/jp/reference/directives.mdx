# ディレクティブ

## #tempo

```plaintext
#tempo(\s|\t)[0-9]+
```

すべてのパートにおいてテンポを設定します。
例えば、拍子の分母が4に設定されているとき、`#tempo 120`を評価すると、四分音符の長さは0.5秒になります。
テンポは1から1000までの整数値でなければなりません。
テンポは標準で100に設定されています。

## #denom

```plaintext
#denom(\s|\t)[0-9]+
```

すべてのパートにおいて拍子の分母を設定します。
例えば、テンポが120に設定されているとき、`#denom 8`を評価すると、四分音符の長さは1秒になります。
分母の値は1から128までの整数値でなければなりません。
分母は標準で4に設定されています。

## #key

```plaintext
#key(\s|\t)[abcdefg]*(+|-|=)
```

すべてのパートにおいて調を設定します。
例えば、`#key cf+`を評価すると、ロ短調（B mimor key）になります。
