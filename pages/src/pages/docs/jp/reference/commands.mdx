# コマンド

## 音符

```plaintext
[abcdefgr](++|--|+|-|=)?[0-9]*'.'?
```

1. 音名
   - `c`：ド(C)
   - `d`：レ(D)
   - `e`：ミ(E)
   - `f`：ファ(F)
   - `g`：ソ(G)
   - `a`：ラ(A)
   - `b`：シ(B)
   - `r`：休符
2. 臨時記号
   - `++`：ダブルシャープ
   - `--`：ダブルフラット
   - `+`：シャープ
   - `-`：フラット
3. 音価
4. 付点

十二平均律における音符一個を表します。
音名以外は省略可能であり、順に「調に合わせて」「標準音価に合わせて」「無付点として」解釈されます。
音価に「1」を指定すると、全音符としては扱われず、一分音符(二分音符2個分)として扱われます。
また、音価は1から128までの整数値でなければなりません。

## 移調

```plaintext
k[abcdefg]*(+|-|=)
```

移調を行います。
「`k`」「変化記号の付くすべての音名」「変化記号」の順で連ねます。

## 標準音価

```plaintext
l[0-9]+
```

音符コマンドにおいて音価を省略した場合に適用される標準の音価を設定します。
「`l`」「音価」の順で連ねます。
音価は1から128までの整数値でなければなりません。
標準音価コマンドが初めて現れるまでは、標準音価は4に設定されています。

## 音量

```plaintext
v[0-9]+('.'[0-9]+)?
```

音量を設定します。
「`v`」「音量」の順で連ねます。
音量は0.0から1.0の小数値でなければなりません。
音量コマンドが初めて現れるまでは、音量は0.5に設定されています。

## オクターブ

```plaintext
<|>|o[1-8]
```

オクターブの移動を行います。
`<`は1オクターブ下へ移動します。
`>`は1オクターブ上へ移動します。
`o`+`番号`は番号で表されるオクターブへ移動します。
ただし、オクターブは1から8までの間しか移動できません。

## テンポ

```plaintext
t[0-9]+
```

テンポの変更します。
「`t`」「テンポ」の順で連ねます。
例えば、拍子の分母が4に設定されているとき、`t120`を評価すると、四分音符の長さは0.5秒になります。
テンポは1から1000までの整数値でなければなりません。
テンポは標準で100に設定されています。

## ループ

```plaintext
'['.*:?.*']'[0-9]+
```

繰返しを表現します。
繰返したいコマンド列を`[]`で囲い、`]`に続いて「繰り返し回数」を指定します。
コマンド列を`:`で区切ると、`:`の右側が最後の繰返しにだけ評価されなくなります。
繰返しは入れ子にできます。
繰返し回数は1以上の整数値でなければなりません。

## マクロ

```plaintext
!MACRO-IDENTIFIER(\s|\t|\n)
```

マクロを展開します。
「`!`」「マクロ名」「区切り文字」の順で連ねます。
マクロの定義については<Link to="/docs/jp/reference/macro-inst/">こちら</Link>を参照してください。

## 楽器

```plaintext
@INST-IDENTIFIER(\s|\t|\n)
```

楽器を変更します。
「`@`」「楽器名」「区切り文字」の順で連ねます。
楽器の定義については<Link to="/docs/jp/reference/macro-inst/">こちら</Link>を参照してください。
