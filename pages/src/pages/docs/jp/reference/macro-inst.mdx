# マクロと楽器

## マクロ

```plaintext
!MACRO-IDENTIFIER(\s|\t)COMMANDS
```

マクロを定義します。
行頭から「`!`」「マクロ名」「区切り文字」「コマンド列」の順で連ねます。
演奏パートの楽譜中にコマンド列を展開できます。
マクロの展開については<Link to="/docs/jp/reference/commands/">こちら</Link>を参照してください。

## 楽器

```plaintext
@INST-IDENTIFIER\n
(
  \t*([0-9]+('.'[0-9]+)(\s|\t)){6}([0-9]+('.'[0-9]+))?\n
)*
```

楽器を定義します。
一行目は「`@`」「楽器名」「改行」の順で連ね、楽器名を指定します。
二行目以降に「インデント」「振幅」「周波数比」「アタックタイム」「ディケイタイム」「サステインレベル」「リリースタイム」「フィードバック回数」の順に「区切り文字」で区切りながら連ね、オペレータを定義します。
インデントを用いてn分木構造で表します。
楽器の適用については<Link to="/docs/jp/reference/commands/">こちら</Link>を参照してください。
