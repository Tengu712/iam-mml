コマンド/音量

文法

number  = non-negative-float ;
command = "+" | "-" ;
octave  = "v" , number , [ command ] ;

解説

音量を指定・変更するコマンド。

  • numberは音量を指定する。 音量は区間[0, 1]内の実数でなければならない。
  • commandは音量を相対的に変更する方向を指定する。 +である場合、音量は加算される。 -である場合、音量は減算される。 commandが与えられていない場合、音量は絶対的に指定される。

標準でv0.5が評価されている。

音量は常に区間[0, 1]内の実数である。 つまり、v0.5v1+を評価しても、音量は1となる。
