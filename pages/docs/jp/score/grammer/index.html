楽譜/文法

partname   = identifier ;
command    = ? a command ? ;
commands   = { command , [ { space } ] } ;
line       = [ { space } ] , partname , { space } , commands , line-break ;
blank-line = [ { space } ] , line-break ;
score      = { line | blank-line } ;
  • partnameはパート名を定義する。 パート名が同じである行は楽譜が繋がっていると解釈される。 ただし、コード上行番号が若い行が時間的に前の部分となる。
  • commandは各種コマンド。
  • commandsはコマンド列挙部分の文法。
  • lineは一行全体の文法。
  • blank-lineは空白行。
  • scoreは楽譜定義エリア全体の文法。

セミコロン「;」以降行末まではコメントとして解釈される。
